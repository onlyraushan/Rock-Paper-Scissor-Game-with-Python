# Rock-Paper-Scissor-Game-with-Python
Rock Paper Scissors – Python Game 🎮
This is a simple Rock, Paper, Scissors game built using Python. The user plays against the computer, which randomly selects one of the options. The program compares both choices and declares the winner based on standard rules.
Features

Command-line based game
Random computer choice
Beginner-friendly logic using conditionals

How to Run

Clone the repository
Run the Python file
Enter your choice when prompted

Requirements

Python 3

Enjoy the game! 😊
