# Rock Paper Scissors – Python Game 🎮

This is a simple **Rock, Paper, Scissors** game built using Python. The game is played in the command line where the user competes against the computer. The computer randomly selects its choice using Python’s `random` module.

## 📌 Rules
- Rock beats Scissor
- Scissor beats Paper
- Paper beats Rock
- Same choice results in a tie

## ✨ Features
- User vs Computer gameplay
- Random computer selection
- Simple and beginner-friendly logic
- Clear match result output

## 🛠 How to Run
1. Clone or download the repository
2. Open terminal / command prompt
3. Run the Python file
4. Enter your choice: Rock, Paper, or Scissor

## 🧾 Requirements
- Python 3.x

## ✅ Sample Output
User Choice = Rock, Computer Choice = Scissor  
You Win  
Have a Good Day

## 📚 Learning Purpose
This project helps beginners understand:
- Conditional statements
- User input
- Random module
- Basic Python programming

Enjoy the game! 😊
